# ⚙ Installation

Clone Repository

```bash
git clone https://github.com/yourusername/AI-Resume-Roaster.git
```

Move to Project

```bash
cd AI-Resume-Roaster
```

Install Dependencies

```bash
npm install
```

Start Development Server

```bash
npm run dev
```

Backend

```bash
cd server

npm install

npm run dev
```

# 🔑 Environment Variables

Create a `.env` file.

```env
VITE_GEMINI_API_KEY=

VITE_FIREBASE_API_KEY=

VITE_FIREBASE_AUTH_DOMAIN=

VITE_FIREBASE_PROJECT_ID=

VITE_FIREBASE_STORAGE_BUCKET=

VITE_FIREBASE_MESSAGING_SENDER_ID=

VITE_FIREBASE_APP_ID=
```

# 📈 How It Works

1. Upload your resume.
2. AI extracts text from the file.
3. Gemini AI analyzes the resume.
4. ATS Score is calculated.
5. HR feedback is generated.
6. Resume improvements are suggested.
7. Generate cover letter.
8. Compare with job description.
9. Download report.

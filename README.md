# 📱 Overview

AI Resume Roaster is an Android application that helps students, freshers, and professionals improve their resumes using Google Gemini AI.

The app evaluates resumes like an ATS (Applicant Tracking System) and HR recruiter, identifies missing skills and keywords, suggests improvements, and helps users prepare for job applications.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Bhavesh12662/ai-resume-roaster.git
```

Open the project

```text
Android Studio
File → Open
Select AI-Resume-Roaster
```

Sync Gradle and run the app.

---

# 🔑 Environment Setup

Create a `.env` file or configure your Gemini API key according to the project setup.

Example:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Never commit your API key to GitHub.
